Why TermiPet?

🎲

Deterministic Gacha

Your pet is seeded from your user ID using Mulberry32 PRNG — same user always hatches the same species, but unique to you.

🎓

Educational

Learn PRNG algorithms, CLI design, event systems, and theming through a fun, interactive project.

🎨

Fully Themeable

Copy themes/example-theme.js and customize colors, borders, stat bars, and messages.

✨

Shiny Variants

Every species has a 1% shiny chance, completely independent of rarity. A Shiny Legendary Nebulynx = 0.01%!

🏗️

Clean Architecture

Modular structure with separate PRNG, Hatchery, Events, Renderer, and CLI layers. Easy to extend.

🖥️

Pure Terminal

No Electron, no browser — just Node.js and chalk. Lives where you work: the terminal.

Get Started

# Clone and install
$ git clone https://github.com/anshmajumdar121/termipet.git
$ cd termipet
$ npm install && npm link

# Hatch your pet!
$ termipet hatch

Commands

termipet hatchHatch your first TermiPet (deterministic — unique to you!)
termipet showDisplay your current pet with ASCII art
termipet watchWatch your pet animate live in the terminal
termipet statsShow detailed stats: DEBUGGING, PATIENCE, CHAOS, WISDOM, SNARK
termipet feedFeed your pet 🍖
termipet playPlay with your pet 🎮
termipet themeSwitch between custom themes
