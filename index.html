Why TermiPet?

🎲

Deterministic Gacha

Seeded from user ID using Mulberry32 PRNG — unique but consistent.

🎓

Educational

Learn PRNG, CLI design, event systems through fun interactive pets.

🎨

Fully Themeable

Customize colors, borders, stat bars, messages easily.

✨

Shiny Variants

1% shiny chance independent of rarity — ultra rare sparkles!

🏗️

Clean Architecture

Modular PRNG, Hatchery, Events, Renderer layers.

🖥️

Pure Terminal

No Electron — just Node.js + chalk. Lives in your CLI.

Get Started

$ git clone https://github.com/anshmajumdar121/termipet.git
$ cd termipet
$ npm install && npm link

$ termipet hatch

Commands

termipet hatchHatch your first TermiPet (deterministic)
termipet showDisplay your pet with ASCII art
termipet watchAnimate live in terminal
termipet statsShow detailed stats
termipet feed / playInteract with pet
termipet themeSwitch themes
